Nameless is a student project, develop in node js, which aims to facilitate the launch of a project (Ionic, Angular, React, React-native) with a terminal.

# Requirements

- Node js v6+ ( https://nodejs.org/en/ )
- Un terminal

### Installation



Install.

```sh
$ git clone https://github.com/AndreasArnolfo/Node-Tool.git
$ cd Node-Tool
$ npm install
```

For start.

```sh
$ cd Node-Tool
$ node . 
```
